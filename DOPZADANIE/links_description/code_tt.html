Описание

Элемент <tt> (от англ. teletype text — текст телетайпа) отображает текст моноширинным шрифтом.

Элемент <code> (от англ. code — код) предназначен для отображения одной или нескольких строк текста,
который представляет собой программный код. Сюда относятся имена переменных, ключевые слова, тексты
функции и т. д. Браузеры обычно отображают содержимое контейнера <code> как моноширинный текст уменьшенного размера.

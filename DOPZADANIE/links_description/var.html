Описание

Элемент <var> (от англ. variable — переменная) используется для выделения переменных из компьютерных программ.
Браузеры обычно помечают текст в контейнере <var> курсивным начертанием.

В данном примере делается запрос к базе данных для получения поля <var>content_title</var>
